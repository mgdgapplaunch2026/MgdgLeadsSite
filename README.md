# MGDG Institutional Portal

## Overview
MGDG is a high-fidelity digital interface providing 1:1 physically allocated gold storage. 
This portal serves as the primary access point for the Pre-Launch Beta Phase (Feb 2025).

## Technical Stack
- **Engine:** Node.js / Express
- **Frontend:** Vanilla JS / CSS3 (Institutional Framework)
- **Typography:** Inter Sans-Serif
